## 1. Executive Summary

### Vision
Build an intelligent trade compliance and cost optimization platform that leverages AI to analyze product descriptions, automate HS code classification, calculate total landed costs, and recommend optimal import/export routing strategiesâ€”reducing duty exposure by 15-30% while ensuring full regulatory compliance.

### Value Proposition
- **For Importers/Exporters**: Reduce landed costs, avoid compliance penalties, accelerate customs clearance
- **For Customs Brokers**: Automate classification work, reduce manual errors, scale operations
- **For Enterprises**: Real-time duty impact analysis for supply chain decisions

### Market Opportunity
- Global trade compliance software market: $1.2B (2024) â†’ $2.8B (2028)
- 95% of SMB importers still use manual classification methods
- Average customs penalty: $50,000-$500,000 per incident

---

## 2. Problem Statement

### Current Pain Points
1. **Manual HS Code Classification**: Time-consuming, error-prone, requires specialized expertise
2. **Unpredictable Landed Costs**: Hidden duties, fees, and taxes discovered at the border
3. **Compliance Complexity**: 200+ FTAs, constantly changing tariff schedules, country-specific rules
4. **Missed Savings**: Companies overpay duties due to suboptimal routing and classification
5. **Penalty Risk**: Misclassification fines average $10,000-$100,000 per incident

### Root Causes
- Lack of AI-powered classification tools
- No unified view of global tariff schedules
- Limited access to FTA optimization strategies
- Siloed data across supply chain systems

### 4.4 Route Optimization Engine

**Feature Description**:
Recommend optimal shipping routes and transshipment points to minimize total costs.

**Optimization Algorithm**:

```python
class RouteOptimizer:
    """
    Multi-objective optimization for trade routes
    """

    def optimize(self, shipment: Shipment) -> List[RouteOption]:
        """
        Find optimal routes considering:
        - Total landed cost
        - Transit time
        - Compliance risk
        - Carbon footprint
        """

        # Generate candidate routes
        routes = self._generate_routes(
            origin=shipment.origin,
            destination=shipment.destination,
            constraints=shipment.constraints
        )

        # Score each route
        scored_routes = []
        for route in routes:
            score = self._calculate_route_score(route, shipment)
            scored_routes.append((route, score))

        # Pareto-optimal selection
        pareto_optimal = self._pareto_filter(scored_routes)

        return sorted(pareto_optimal, key=lambda x: x[1].total_cost)

    def _calculate_route_score(self, route: Route, shipment: Shipment) -> RouteScore:
        return RouteScore(
            total_cost=self._calc_landed_cost(route, shipment),
            transit_days=self._calc_transit_time(route),
            compliance_risk=self._assess_compliance_risk(route),
            carbon_kg=self._calc_carbon_footprint(route, shipment),
            fta_savings=self._calc_fta_benefit(route, shipment)
        )
```

    ruling_reference VARCHAR(100),

    created_by UUID REFERENCES users(id),
    created_at TIMESTAMPTZ DEFAULT NOW(),
    confirmed_at TIMESTAMPTZ
);

CREATE INDEX idx_classifications_hs ON classifications(hs_code_6);
CREATE INDEX idx_classifications_org ON classifications(org_id);

-- Tariff Schedules (versioned)
CREATE TABLE tariff_schedules (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    country_code CHAR(2) NOT NULL,
    hs_code VARCHAR(15) NOT NULL,
    description TEXT,
    duty_rate DECIMAL(10,4),
    duty_type VARCHAR(20), -- 'ad_valorem', 'specific', 'compound'
    specific_rate DECIMAL(10,4),
    specific_unit VARCHAR(20),
    effective_date DATE NOT NULL,
    expiry_date DATE,
    source VARCHAR(100),
    additional_duties JSONB DEFAULT '[]',
    created_at TIMESTAMPTZ DEFAULT NOW(),

    UNIQUE(country_code, hs_code, effective_date)
);

CREATE INDEX idx_tariff_country_hs ON tariff_schedules(country_code, hs_code);
CREATE INDEX idx_tariff_effective ON tariff_schedules(effective_date);

-- FTA Rules
CREATE TABLE fta_agreements (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    code VARCHAR(20) UNIQUE NOT NULL, -- 'USMCA', 'RCEP', etc.
    name VARCHAR(255) NOT NULL,
    member_countries CHAR(2)[] NOT NULL,
    effective_date DATE,
    status VARCHAR(50) DEFAULT 'active',
    documentation_url TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE fta_product_rules (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    fta_id UUID REFERENCES fta_agreements(id),
    hs_code_pattern VARCHAR(15) NOT NULL, -- supports wildcards
    rule_type VARCHAR(50), -- 'tariff_shift', 'rvc', 'process', 'wholly_obtained'
    rule_detail JSONB NOT NULL,
    preferential_rate DECIMAL(10,4),
    notes TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Landed Cost Calculations
CREATE TABLE landed_cost_calculations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    org_id UUID REFERENCES organizations(id),
    classification_id UUID REFERENCES classifications(id),

    -- Input parameters
    origin_country CHAR(2) NOT NULL,
    destination_country CHAR(2) NOT NULL,
    destination_port VARCHAR(10),
    product_value DECIMAL(15,2) NOT NULL,
    currency CHAR(3) DEFAULT 'USD',
    quantity INTEGER DEFAULT 1,
    weight_kg DECIMAL(10,3),
    incoterm VARCHAR(3),
    shipping_mode VARCHAR(20),

    -- Calculated results
    total_landed_cost DECIMAL(15,2),
    base_duty DECIMAL(15,2),
    fta_savings DECIMAL(15,2),
    additional_duties DECIMAL(15,2),
    taxes DECIMAL(15,2),
    fees DECIMAL(15,2),
    logistics DECIMAL(15,2),

    -- Metadata
    calculation_breakdown JSONB,
    fta_applied VARCHAR(20),
    confidence_score DECIMAL(5,2),
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Audit Trail
CREATE TABLE audit_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    org_id UUID REFERENCES organizations(id),
    user_id UUID REFERENCES users(id),
    action VARCHAR(100) NOT NULL,
    entity_type VARCHAR(50),
    entity_id UUID,
    changes JSONB,
    ip_address INET,
    user_agent TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_audit_org_date ON audit_logs(org_id, created_at DESC);
